# Weather Tracker Application

A real-time weather tracking application built with .NET 8, React, and Azure cloud services.

## Technologies Used

### Backend
- .NET 8
- Azure Container Apps
- Azure Functions
- OpenWeatherMap API

### Frontend
- React with TypeScript
- Material UI
- Axios

### DevOps
- Docker
- GitHub Actions
- Azure Container Registry

## Getting Started

### Prerequisites
- .NET 8 SDK
- Node.js
- Docker Desktop
- Azure CLI
- OpenWeatherMap API Key
